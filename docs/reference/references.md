# Glossary

## External Resources

### AYAB Project

**[AYAB - All Yarns Are Beautiful](https://ayab-knitting.com/)**

The AYAB project provides open-source solutions for computerized knitting machine control. This firmware implements the AYAB protocol to bring the same capabilities to Silver Reed machines.

- **[AYAB Desktop Software](https://ayab-knitting.com/)** - Pattern creation and knitting control software
- **[AYAB API Documentation](https://manual.ayab-knitting.com/1.0/development/API/)** - Protocol specifications
- **[AYAB GitHub](https://github.com/AllYarnsAreBeautiful)** - Source code and development
